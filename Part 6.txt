반복문

while
조건을 먼저 물어보고 조건이 맞으면 실행문을 실행
if 문과 마찬가지로 중괄호가 없다면 실행문이 한줄로 파악되어 실행문을 실행함
-> 의도와 다르게 적용될 가능성이 있음
또한 while(조건) 과 안에 있는 실행문을 한 문장으로 파악하기 때문에
if (조건) 
    while (조건){
        실행문
    }
과 같이 사용해도 문제없이 실행된다.


for
초깃값을 최초로 설정한 다음 조건을 물어보고 실행문을 실행한 다음 증감식을 실행후 조건을 확인;;;
while과 비슷함 ; 보통 횟수가 정해져있게 반복을 할때 사용함

주의사항; 
초기식, 조건식, 증감식을 알기 쉽게 하는 것이 좋음
반복횟수를 세는 숫자; i를 바꾸지 않는 것이 좋음


do while
실행문을 먼저 실행하고 조건을 확인
while과 달리 조건을 확인하지 않고 실행문을 먼저 실행하고 조건을 확인함
do {
    실행문
} whlie (조건식)


반복문 활용
중첩 반복문;
반복문 안에 반복문을 집어넣음


break continue 분기문
break ; 반복문 안에 사용하면 반복을 즉시 끝냄
조건식에 예외적인 경우를 상정하여 반복문을 끝낼 수도 있음

continue; 반복문의 일부를 뛰어넘음
continue를 실행하면 다음 실행위치가 반복문의 블록 끝이됨.
블록을 탈출하는 것이아님 -> 조건에 따라 반복문의 일부를 제외하고 반복할 수 있음
