변수

변수 선언 방법;
int a;
자료형 변수명의 형태임
int == 4byte

저장할 공간을 a라고 이름을 짓고 메모리에 4바이트를 할당하겠다.고 컴파일러한테 알려주는 것
        변수명은 임시 주소

초기화를 안하면 메모리에 쓰레기 값이 들어가므로, 초기화를 시켜야함
a = 10; 10

변수는 데이터를 저장하는 메모리의 공간. 데이터에 따라 다양한 형태를 사용함
int, char, float 등등

정수 자료형
char, short, int, long, long long이 있음
1     2      4    4     8바이트
컴파일러에 따라 int가 2byte인 경우도 있음
sizeof() 로 알아볼 수 있다.

char에는 일반적으로 문자를 집어넣음
초기화될때 문자를 집어넣는 경우 아스키 코드값에 해당하는 숫자로 초기화됨;
이는 다른 자료형에도 마찬가지임

unsigned 자료형
음수가 없는 자료형으로 조금더 큰 정수를 입력할 수 있음


실수 자료형
float, double, long double
4      8       8이상
7      15      15이상 (유효 소숫점)

실수형은 초기화 하는 값에 따라 숫자가 정확하게 저장될 수 도 있고, 그렇지 않을 수도 있음


문자열 저장
char 배열명[문자열길이 + 1] = 문자열;
"apple\0" 의 형태로 배열에 들어가게됨

출력은 %s

초기화 이외에 값을 변경하려면 대입연산자가 아니라 string.h 헤더파일을 추가하고
strcpy(배열명, "문자열")로 바꿀수 있음


const 를 사용한 변수
변수는 저장공간이므로 언제든지 그 값을 변경할 수 있지만, const를 사용하면 초기화된 값을 변경할 수 없음;;;;
또한 const는 선언과 동시에 초기화를 해주어야함
const int avr = 11;
;쓰레기 값이 들어간 상태로 값을 변경할 수 없게 된다는 건 말이 안됨


예약어와 식별자
예약어는 컴파일러와 사용방법이 약속된 단어
식별자는 필요에 따라 만들어 사용하는 단어 ( 우리가 )

식별자를 정하는 규칙이 있음
1. 알파벳 A~Z, a~z, 숫자 0~9, _
2. 숫자로 시작 x
3. 대문자와 소문자는 서로다른 식별자
4. 예약어는 식별자로 지정불가
